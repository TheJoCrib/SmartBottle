# Smart Vätskemätare för Flaska - React Native x Expo

Ett smart system designat för att mäta mängden vätska i din flaska före och efter användning, och på ett intelligent sätt meddela dig om din vätskestatus.

## Översikt

Detta projekt gör det möjligt för en Arduino att beräkna mängden vätska som finns kvar i en flaska och kommunicera denna information till användaren på ett smart och bekvämt sätt. Oavsett om du vill hålla koll på ditt dagliga vattenintag, övervaka dina vätskevanor eller helt enkelt veta när det är dags att fylla på — så har det här systemet dig täckt.

## Funktioner

- **Noggrann vätskemätning** — Beräknar mängden vätska i flaskan före och efter konsumtion
- **Smarta notiser** — Få meddelanden i rätt tid om din vätskestatus
- **Realtidsuppföljning** — Övervaka ditt vätskeintag under dagen
- **Användarvänligt gränssnitt** — Enkel och tydlig återkoppling

## Så fungerar det

1. Systemet mäter den ursprungliga mängden vätska i din flaska
2. Efter att du har druckit beräknas den nya kvarvarande mängden
3. Skillnaden registreras som din konsumtion
4. Smarta notiser informerar dig om dina framsteg och påminner dig om att dricka mer!
